## **5. Instructions for running:**
All you need to do is run it, once you run it first you need to put the number of cases you want to run, followed by the cases you need to put an integer, which means the number of non-terminal symbols that your grammar has, followed by this you need to put your grammar. The program will return the "First" set and the "Follow" set for each grammar you entered, according to the number of cases you entered.
